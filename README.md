# Stone-Paper-Game
he Stone Paper Scissors game (also known as Rock Paper Scissors) is a simple and fun hand game usually played between two players. In this project, the user plays against the computer. Both choose one option — Stone, Paper, or Scissors — and the winner is decided based on predefined rules.
# Game Rules

 Stone beats Scissors

 Scissors beats Paper

 Paper beats Stone

# Features

User vs Computer gameplay

Random computer choice

Instant result display (Win / Lose / Draw)

Simple and beginner-friendly logic
